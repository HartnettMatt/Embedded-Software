void add_scheduled_event ( uint32_t  event)

Function to add an event to the scheduler.

This function adds the event to the scheduler through an or-equals function to the static variable with the event integer. This ensures that other events are not effected by the adding of a new event.

Note
This function is atomic to prevent issues with interrupts changing the event scheduled bit multiple times.
Parameters
[in]eventAn integer that represents an event that needs to be serviced.

◆ remove_scheduled_event()

void remove_scheduled_event ( uint32_t  event)

Function to remove an event to the scheduler.

This function removes the event to the scheduler through an and-not-equals function to the static variable with the event integer. This ensures that other events are not effected by the removing of a new event.

Note
This function is atomic to prevent issues with interrupts changing the event scheduled bit multiple times.
Parameters
[in]eventAn integer that represents an event that needs to be serviced.

void scheduler_open ( void  )

Driver to open a scheduler for event handlers.

Initializes the scheduler to have no events scheduled, by utilizing a static variable that keeps track of which events need to be serviced.

Note
This function is atomic to prevent issues with interrupts changing the event scheduled bit multiple times.
