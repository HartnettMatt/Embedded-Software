void si7021_i2c_open ( uint32_t  si7021_read_cb)

This function opens the I2C peripheral with the correct values need to read a value from the SI7021.

This function calls the i2c_open() function, and uses the definitions in si7021.h to ensure that the I2C peripheral is configured to correctly read a value from the si7021.

Parameters
[in]si7021_read_cbThe scheduler event value, which is required to clear the scheduler when the I2C state machine has terminated.

This function initiates a read of the si7021 over the I2C bus.

This function calls the i2c_start() function, and uses the definitions in si7021.h to ensure that the I2C peripheral is configured to correctly read a value from the si7021.

float si7021_temp ( void  )

This function converts the temperature data code from the si7021 to the temperature in Fahrenheit.

This function uses the equation found in the si7021 documentation to translate the code from the si7021 peripheral into a temperature on a more useful scale.
